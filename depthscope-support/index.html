Quick Support Notes

Useful for App Store support and first-line troubleshooting

Supported hardware

  • LiDAR-capable iPhone or iPad
  • Rear camera access enabled
  • Apple Vision Pro and Mac distribution are intentionally disabled

When contacting support

  • Describe the device model and iOS or iPadOS version
  • Describe what step failed: capture, processing, or export
  • Include screenshots if they do not expose sensitive content

Support / 日本語

DepthScope サポート案内

概要 DepthScope は LiDAR 対応 iPhone / iPad 向けの深度比較アプリです。サポートが必要な場合は、このページの公開リンクからお問い合わせください。

1. サポート窓口

  • GitHub Issues に問い合わせ内容を投稿する
  • 開発者 GitHub プロフィール を確認する

2. 対応端末

本アプリは LiDAR センサーを搭載した iPhone / iPad を前提にしています。LiDAR 非対応端末では、想定どおり動作しない、または一部機能が使えない場合があります。

3. よくある確認ポイント

  • カメラ権限が有効になっているか
  • 背面カメラと LiDAR を使える端末か
  • 処理後のレポートを共有シート経由でエクスポートしているか

4. 問い合わせ時に含めてほしい情報

  • 端末名と OS バージョン
  • どの操作で問題が発生したか: 撮影、処理、レポート表示、エクスポート
  • 表示されたエラーメッセージ
  • 必要に応じてスクリーンショット

Support / English

DepthScope support information for App Store users

Summary DepthScope is a depth comparison app for LiDAR-capable iPhone and iPad devices. Use the public links on this page if you need help.

1. Support channels

  • Open a request through GitHub Issues
  • Check the public developer profile at github.com/miyachin-paperdriver

2. Supported devices

This app is intended for iPhone and iPad devices that include a LiDAR sensor. On unsupported hardware, the app may not function as intended or some features may be unavailable.

3. Common first checks

  • Confirm that camera access is allowed
  • Confirm that the device supports the rear camera and LiDAR workflow required by the app
  • Confirm that report export is being performed through the system share sheet

4. Information to include in a support request

  • Device model and OS version
  • Which step failed: capture, processing, report display, or export
  • The error message shown by the app, if any
  • Screenshots when appropriate and safe to share
